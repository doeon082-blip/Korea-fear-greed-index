# 한국 공포탐욕지수(Korea-Fear-Greed-Index)

## 프로젝트 개요
코스피 데이터를 기반으로 한국 주식시장의 투자 심리를 0에서 100으로 cnn 공포탐욕지수를 참고하여 한국 시장에 맞게 재설계하였습니다

## 만들게 된 계기
제가 주식 투자의 고민을 하고 본격적인 투자를 시작할 무렵 워렌 버핏이 공포 탐욕 지수 를 본다는 말을 들었습니다 근데 이것도 cnn 외국 사이트라 접근성이나 처음에 찾기는 쉽지 않았고 그리고 우리 나라가 미국 시장의 영향을 받는다고는 하여도 한국시장의 맞는 지표가 부족할뿐 아니라 VKOSPI도 접근성이 좋치 않아서 한동 cnn 공포탐욕지수를 보고 불편함을 느껴서 만들고 싶다는 생각이 들엇습니다 아직은 부족하나 점차 업데이트를 하고자 하며 아직 비전공자라 ai의 도움을 받으면서 같이 공부하고 있으면 독학중입니다 목표는 ai 대학원의 진학을 희망하여 나중에는 llm이랑도 연동하여 더욱더 퀄리티를 올릴생각입니다 

## 발전가능성
아직 이건 기본 지수로만 보여주는것이며 추후 llm 기술을 접목시켜 이 프로젝트를 확장 할려고 합니다 
또한 코스피뿐만 아니라 변동성이 강한 코스닥 미국 시장인 나스닥 s&p500 투자 심리도 만들어서 접목 시켜 한국 주식의 투자자들이 좀더 좋은족으로 투자를 할수있는게 제 목표이자 최종 이 프로젝트의 목표라고 생각합니다 

## 현재 한계및 차별점

### 한계
- 외국인 순매수 데이터 미반영 (현재 매수강도로 대체)
- 지표 가중치 동일 (추후 XGBoost로 최적화 예정)

### 기존 지표와의 차별점
- CNN Fear & Greed는 미국 시장 기반
- 본 프로젝트는 한국(KOSPI) 시장에 맞게 재설계
- 일반 투자자가 직관적으로 읽을 수 있도록 0~100으로 정규화

## 사용 기술
- Python
- Streamlit
- FinanceDataReader
- Pandas
- Matplotlib

## 7개 지표
1. 이동평균 괴리율 (MA Gap)
2. 거래량 변화율 (Volume Change)
3. 변동성 (Volatility)
4. 모멘텀 (Momentum)
5. 52주 신고가/신저가 비율 (High-Low Ratio)
6. 매수 강도 (Buy Pressure)
7. RSI

## 결과 해석
| 점수 | 상태 |
|------|------|
| 75 이상 | 극도의 탐욕 |
| 55 ~ 74 | 탐욕 |
| 45 ~ 54 | 중립 |
| 25 ~ 44 | 공포 |
